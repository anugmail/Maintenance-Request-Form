# 🗺️ Diagram — รวม Flow ทุกอันของโปรเจกต์ (Mermaid)

รวม **flowchart ของทุก flow** ในโปรเจกต์ Maintain-D แยกเป็นไฟล์ย่อยชัดเจน — เปิดดูเป็นภาพได้ง่าย

## 👁️ ดูเป็นภาพยังไง (3 วิธี)
1. **GitHub (ง่ายสุด)** — เปิดไฟล์ `.md` ใดๆ ในโฟลเดอร์นี้บน GitHub → **เห็นเป็นภาพทันที** (GitHub render ` ```mermaid ` ให้อัตโนมัติ ไม่ต้องลงอะไร)
2. **VS Code** — เปิดไฟล์ → กด Preview (`Cmd/Ctrl+Shift+V`); ถ้าไม่ขึ้นให้ลง extension **"Markdown Preview Mermaid Support"**
3. **mermaid.live** — ก็อปโค้ดในบล็อก ` ```mermaid ` ไปวางที่ https://mermaid.live เห็นภาพ + export PNG/SVG ได้

> ทำไม Mermaid ไม่ใช่ PlantUML? — GitHub render Mermaid ให้ฟรี (เห็นภาพทันทีบนเว็บ) ส่วน PlantUML ต้องมี tool/แปลงเป็นรูปก่อนถึงจะเห็น
