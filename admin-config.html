แผงควบคุมต้นแบบ (Admin Control Panel)

ปรับธีม · สลับรูปแบบ UI · เปิด/ปิดฟีเจอร์ · เติมข้อมูลเดโม · จัดการข้อมูลหลัก — โดยไม่ต้องแก้โค้ด
open_in_newเปิดต้นแบบ
lightbulbเปิดหน้านี้คู่กับหน้าต้นแบบในอีกแท็บ — เปลี่ยนธีมเห็นผลทันทีไม่รีโหลด ส่วนรูปแบบ UI / ฟีเจอร์ / ข้อมูล / เดโม หน้าต้นแบบจะรีโหลดอัตโนมัติ · ค่าทั้งหมดจำไว้ในเบราว์เซอร์นี้ (localStorage) แม้ปิดเปิดใหม่

1 · ปรับแต่งดีไซน์ (Theme)

สี ฟอนต์ มุมโค้ง มีผลกับทุกหน้าจอผ่าน design tokens — การ์ดนี้เป็นตัวอย่างสด ปรับแล้วหน้านี้เปลี่ยนตามด้วย
ชุดสีแบรนด์
ระบบคำนวณเฉดอ่อน/เข้มให้อัตโนมัติ
ขนาดตัวอักษร
ความโค้งมุม (ปุ่ม/การ์ด)
ตัวอย่าง component ตามธีมปัจจุบัน สถานะเรื่อง พร้อมเบิก
ชิพที่เลือก
ชิพปกติ

2 · รูปแบบ UI (Design Variants)

สลับดีไซน์ของจุดต่าง ๆ ในต้นแบบ เพื่อเทียบสไตล์กับผู้ใช้ — เช่น การ์ดผืนผ้า ↔ จตุรัส หรือสไตล์การเลือกวันนัด

3 · ควบคุมการเดโม (Demo)

เตรียมสถานการณ์ก่อนพรีเซนต์ — เลือกหน้าเริ่มต้น และเติมชุดงานซ่อมตัวอย่าง (แทนที่งานในต้นแบบทั้งชุด)
หน้าเริ่มต้นเมื่อเปิดต้นแบบ
ชุดข้อมูลงานซ่อม
* งานที่สร้างสด ๆ ในต้นแบบเป็นข้อมูลชั่วคราว หายเมื่อรีโหลด — ชุดข้อมูลที่เลือกที่นี่จะถูกใช้ทุกครั้งที่เปิดต้นแบบ

4 · เปิด/ปิดฟีเจอร์ (Feature Toggles)

ซ่อนเมนู ขั้นตอน หรือฟิลด์ เพื่อลองเทียบ flow แบบมี/ไม่มีฟีเจอร์นั้นกับผู้ใช้

5 · จัดการข้อมูลหลัก (Master Data)

แก้ไขรายการ รถ / อะไหล่ / อู่ ที่ต้นแบบใช้ — แก้ในตารางแล้วกด "บันทึก" ของแต่ละชุด (หน้าต้นแบบจะรีโหลดใช้ข้อมูลใหม่)
local_shippingรถ แก้ไขแล้ว — ยังไม่บันทึก
settingsอะไหล่ แก้ไขแล้ว — ยังไม่บันทึก
garageอู่แนะนำ แก้ไขแล้ว — ยังไม่บันทึก
